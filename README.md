# IC — Volume Profile + Footprint Indicators

Pine Script v6 indicators by **Infusion Capital**.

## PSP PRO — Power Session Pulse PRO

Full-featured indicator combining:
- **Volume Profile** — 50-row histogram (POC, VAH, VAL, Value Area 70%)
- **Session Boxes** — Asia / London / NY with high & low markers
- **Order Flow / Delta** — Tick Rule via lower timeframe (auto: 1H+ → 5min, <1H → 1min)
- **MSS Detection** — Market Structure Shift (bullish & bearish)
- **TRAP Detection** — Failed MSS reversal signals
- **Confluence Scoring** — A+ / B / C grades (MSS + Delta anomaly + VA break + Session break)
- **Dashboard** — Bull BO Count, Bear BO Count, TP Hits, Fake Outs, Delta %

## IC VP+FP Gravity

Simpler version — Volume Profile + Footprint Delta panel (original build).

## Usage

1. Open TradingView Pine Editor
2. Paste the contents of `PSP_PRO.pine`
3. Click **Add to chart**
